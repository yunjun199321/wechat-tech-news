# 焦点板块模板与最佳实践

> **用途**: 为微信公众号科技新闻文章创建吸引人的"48小时焦点"板块
>
> **目标**: 在文章开头快速抓住读者注意力,提高完读率和分享率

---

## 标准格式

```markdown
## 🌟 48小时焦点

> 快速浏览48小时最重要的科技动态

1. **🇺🇸 [地域标识] [8-12字精炼标题]**
   [25-35字一句话核心信息,突出影响和意义]

2. **🇨🇳 [地域标识] [8-12字精炼标题]**
   [25-35字一句话核心信息,突出影响和意义]

3. **🌍 [地域标识] [8-12字精炼标题]**
   [25-35字一句话核心信息,突出影响和意义]

4. **[地域标识] [标题]**
   [一句话总结]

5. **[地域标识] [标题]**
   [一句话总结]
```

## 选择标准(加权评分)

### 评分系统

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **影响范围** | 30% | 全球性(10分) > 区域性(7分) > 单一市场(4分) |
| **读者相关度** | 25% | 直接影响(10分) > 行业影响(7分) > 一般兴趣(4分) |
| **新颖性** | 20% | 突发新闻(10分) > 重大更新(7分) > 预期内(4分) |
| **地域平衡** | 15% | 平衡国内外(10分) > 偏向一方(6分) > 单一地域(3分) |
| **争议程度** | 10% | 适度争议(10分) > 无争议(7分) > 高度敏感(3分) |

**使用方法**:
1. 为每条新闻的5个维度评分
2. 计算加权总分
3. 选择总分最高的5条新闻
4. 确保地域平衡(至少2条国内,2条国外)

### 高分新闻类型

#### 🏆 必选类型(通常90分以上)

1. **重大政策变化**
   - 政府投资计划(数十亿美元级)
   - 产业政策调整
   - 监管法规出台

   **示例**: "美国政府6亿美元AI研发计划"

2. **技术重大突破**
   - 量子计算商用
   - 芯片制程突破
   - AI能力跨越

   **示例**: "量子计算进入商用元年"

3. **市值/融资里程碑**
   - 万亿级投资计划
   - 千亿美元市值突破
   - 超大型融资轮次

   **示例**: "NVIDIA市值突破5万亿美元"

4. **行业格局变化**
   - 重大并购交易
   - 市场份额巨变
   - 产业链重构

   **示例**: "七大科技巨头万亿投资计划启动"

#### ✅ 优选类型(通常80-89分)

1. **产品重大发布**
   - 革命性新产品
   - 重要功能更新
   - 平台级应用

   **示例**: "OpenAI发布多模态AI助手"

2. **企业战略调整**
   - 业务重组
   - 组织架构变化
   - 战略合作

   **示例**: "微软AI业务独立运营"

3. **人才重大变动**
   - 高层人事调整
   - 关键人才加盟
   - 行业人才流动

   **示例**: "前Google AI负责人加盟OpenAI"

4. **行业趋势显现**
   - 新兴应用爆发
   - 技术范式转变
   - 商业模式创新

   **示例**: "AI代理应用爆发式增长"

#### ⚠️ 谨慎选择(通常70分以下)

1. **常规财报** (除非数据特别亮眼)
2. **小规模融资** (低于1亿美元)
3. **预期内的进展** (缺乏新闻性)
4. **高度敏感话题** (合规风险高)

## 地域标识使用

### 标识系统

| Emoji | 含义 | 使用场景 |
|-------|------|----------|
| 🇨🇳 | 中国 | 中国大陆企业、港澳台企业、在华外企 |
| 🇺🇸 | 美国 | 美国企业、美国政策 |
| 🇪🇺 | 欧盟 | 欧盟企业、欧盟政策 |
| 🌍 | 全球 | 跨国合作、全球趋势、国际组织 |

### 特殊情况处理

**跨国企业在华业务**:
- 特斯拉上海工厂 → 🇨🇳 (业务发生地)
- 苹果中国政策 → 🇨🇳 (影响地域)

**国际合作项目**:
- 中美企业合作 → 🌍 (跨国性质)
- 全球气候技术联盟 → 🌍 (国际组织)

**政策影响**:
- 美国芯片政策影响中国 → 🇺🇸 (政策制定地)
- 欧盟AI法案 → 🇪🇺 (法案归属地)

## 标题创作技巧

### 长度控制

**最佳长度**: 8-12字
**最大长度**: 不超过15字
**最小长度**: 不少于6字

### 标题公式

#### 公式1: 主体 + 金额 + 行动
```
政府 + 6亿美元 + 押注AI
→ "美国政府6亿美元押注AI代理"
```

#### 公式2: 主体 + 数字 + 里程碑
```
NVIDIA + 5万亿 + 市值突破
→ "NVIDIA市值突破5万亿美元"
```

#### 公式3: 主体 + 时间 + 事件
```
量子计算 + 元年 + 商用
→ "量子计算进入商用元年"
```

#### 公式4: 主体 + 规模 + 计划
```
七大企业 + 万亿 + 投资
→ "科技巨头万亿投资计划"
```

### 标题优化对比

| ❌ 避免 | ✅ 推荐 | 原因 |
|---------|---------|------|
| OpenAI获得合同 | 政府6亿美元押注AI | 突出规模和影响 |
| NVIDIA股价上涨 | NVIDIA市值破5万亿 | 强调里程碑 |
| 量子计算有进展 | 量子计算商用元年 | 明确阶段性 |
| 企业要投资AI | 七大巨头万亿计划 | 量化规模 |

## 描述撰写技巧

### 描述结构

**标准结构**: 核心事实 + 影响/意义

```
[核心事实: 15-20字] + [影响/意义: 10-15字]
```

**示例**:
```
美国政府向OpenAI等四家AI公司分配研发合同,标志着政府AI应用进入加速期
└─────────核心事实─────────┘ └────影响/意义────┘
```

### 描述公式

#### 公式1: 谁 + 做了什么 + 意味着什么
```
"七大科技企业未来三年将投资超1万亿元,AI基础设施建设成为行业重点"
```

#### 公式2: 什么 + 达到什么规模 + 反映什么趋势
```
"NVIDIA市值突破5万亿美元,成为全球第三大上市公司,AI芯片需求持续旺盛"
```

#### 公式3: 什么技术 + 什么进展 + 什么应用
```
"IBM推出1000量子比特处理器,商用价格降低60%,金融行业率先应用"
```

### 描述优化对比

| ❌ 避免 | ✅ 推荐 |
|---------|---------|
| OpenAI获得了6亿美元的政府合同用于AI研发 | 美国政府向四家AI巨头分配6亿美元研发合同,政府AI应用进入加速期 |
| NVIDIA的股票表现很好 | NVIDIA市值突破5万亿美元,成为全球第三大上市公司,AI芯片需求旺盛 |
| 很多公司要投资AI | 阿里、腾讯等七大企业未来三年投资超1万亿元,AI基础设施成重点 |

## 完整示例

### 示例1: 平衡型(推荐)

```markdown
## 🌟 48小时焦点

> 快速浏览48小时最重要的科技动态

1. **🇺🇸 政府6亿美元押注AI代理**
   美国政府向OpenAI、Anthropic、Google、xAI四家公司分配AI研发合同,标志着政府AI应用进入加速期

2. **🇨🇳 科技巨头万亿投资计划启动**
   阿里、腾讯、字节、华为等七大企业未来三年将投资超1万亿元,AI基础设施建设成为重点

3. **🌍 NVIDIA市值突破5万亿美元**
   成为全球第三大上市公司,仅次于苹果和微软,AI芯片需求持续旺盛推动增长

4. **🇪🇺 欧盟AI法案正式生效**
   全球首个全面AI监管法规实施,对高风险AI系统提出严格合规要求,影响全球科技企业

5. **🌍 量子计算进入商用元年**
   IBM推出1000量子比特处理器,商用价格降低60%,金融和制药行业率先应用
```

**优点**:
- ✅ 地域平衡(2国内 + 2国外 + 1全球)
- ✅ 主题多样(政策、投资、市值、监管、技术)
- ✅ 数字突出(6亿、万亿、5万亿、1000量子比特、60%)
- ✅ 影响清晰(加速期、重点、推动增长、合规要求、率先应用)

## 常见错误与修正

### 错误1: 标题过长

❌ **错误**:
```
美国政府宣布向OpenAI等四家人工智能公司分配总计6亿美元的研发合同
```

✅ **修正**:
```
政府6亿美元押注AI代理
```

### 错误2: 描述冗长

❌ **错误**:
```
美国政府最近宣布了一项总计6亿美元的人工智能研发合作计划,
将资金分配给OpenAI、Anthropic、Google和xAI这四家主要的AI公司,
这标志着美国政府在人工智能领域的应用正在进入一个新的加速发展期
```

✅ **修正**:
```
美国政府向OpenAI等四家AI公司分配6亿美元研发合同,政府AI应用进入加速期
```

### 错误3: 缺乏数字

❌ **错误**:
```
科技企业加大AI投资力度
```

✅ **修正**:
```
七大科技企业万亿投资计划启动
```

### 错误4: 影响不明

❌ **错误**:
```
NVIDIA市值创新高
```

✅ **修正**:
```
NVIDIA市值突破5万亿美元,成为全球第三大上市公司
```

### 错误5: 地域失衡

❌ **错误**:
```
5条全是美国企业新闻
```

✅ **修正**:
```
至少2条国内 + 2条国外 + 1条全球/欧盟
```

## 合规注意事项

### 敏感话题处理

#### 政府合作项目

❌ **避免**:
```
五角大楼AI军事合同分配
```

✅ **使用**:
```
政府6亿美元AI研发计划启动
```

#### 贸易政策

❌ **避免**:
```
美国禁止向中国出口AI芯片
```

✅ **使用**:
```
AI芯片出口政策调整,产业链加速本地化
```

#### 企业负面新闻

❌ **避免**:
```
科技公司大规模裁员潮来袭
```

✅ **使用**:
```
科技企业组织架构优化调整
```

#### 金融市场

❌ **避免**:
```
科技股暴跌,投资者血亏
```

✅ **使用**:
```
科技股市场调整,行业进入整合期
```

## A/B测试建议

### 测试变量

1. **数量测试**: 3条 vs 5条 vs 7条
2. **顺序测试**: 国内优先 vs 国外优先 vs 影响力排序
3. **emoji测试**: 地域emoji vs 类别emoji vs 混合使用
4. **描述长度**: 20字 vs 35字 vs 50字

### 评估指标

- **滑动深度**: 用户是否滑过焦点板块继续阅读
- **完读率**: 阅读到文章结尾的用户比例
- **分享率**: 分享/阅读比例
- **停留时间**: 焦点板块平均停留时间

## 实战Checklist

创建焦点板块时,逐项检查:

- [ ] 选择了5条高分新闻(评分系统筛选)
- [ ] 地域平衡(至少2国内 + 2国外)
- [ ] 每条标题8-12字
- [ ] 每条描述25-35字
- [ ] 包含关键数字(金额、数量、百分比)
- [ ] 说明了影响和意义
- [ ] 使用了合规语言
- [ ] 添加了地域emoji
- [ ] 主题多样化(不重复)
- [ ] 引导语简洁有力
