# 微信文章标准结尾模板

> **用途**: 为所有微信科技新闻文章提供统一的结尾格式
>
> **包含**: 免责声明、下期预告、订阅提示

---

## 标准结尾格式 (v3.0 精简版)

```markdown
## 📋 免责声明

本报告基于[日期范围]的公开信息整理，数据来源包括官方公告和主流科技媒体（TechCrunch、Bloomberg、Reuters等）。**所有财务数据仅供参考，不构成投资建议。** 技术规格、产品信息及行业预测以公司官方最终公告为准，实际发展可能存在差异。

---

**📅 下期预告**

重点关注：
- [即将发生的重要活动1]
- [即将发生的重要活动2]
- [即将发生的重要活动3]

---

**🔔 订阅提示**

关注本公众号，每日获取精选AI科技新闻，不错过重要动态！
```

**变更说明 (v3.0)**：
- ✂️ 从17行精简到4行（-76% 篇幅）
- ✅ 保留核心法律必需条款（投资免责）
- ✅ 保留数据来源和准确性说明
- ✅ 删除冗余的分段说明（财务/技术/预测）

## 使用说明

### 1. 日期范围

根据文章涵盖的时间范围填写：
- 周报：本报告基于2025年11月12日至11月17日...
- 日报：本报告基于2025年11月17日...
- 月报：本报告基于2025年11月...

### 2. 下期预告

**选择标准**：
- 已确定的重要活动（如大型会议、财报发布）
- 时间在未来1-2周内
- 对科技行业有重大影响
- 3-5项为宜

**示例**：
```markdown
重点关注：
- Microsoft Ignite大会（11月18-21日）
- Alibaba Q2财报（11月25日）
- AI行业年终总结与2026展望
- OpenAI DevDay开发者大会（12月1日）
- NVIDIA GTC大会（12月3-5日）
```

### 3. 媒体来源

根据实际引用的媒体来源调整：

**常用国际媒体**：
- TechCrunch、Bloomberg、CNBC、Reuters
- The Verge、Ars Technica、Wired
- VentureBeat、TechTarget

**常用中文媒体**：
- 36氪、钛媒体、界面新闻
- 财新、第一财经、经济观察报

**示例**：
```markdown
数据来源包括官方公告、主流媒体报道（TechCrunch、Bloomberg、36氪、钛媒体等）和行业分析。
```

## 注意事项

### ✅ 必须包含 (v3.0)

1. **精简但合规的免责声明** - 4行单段，包含投资免责+数据准确性说明
2. **明确的日期范围** - 让读者知道信息的时效性
3. **具体的下期预告** - 3-5项重要活动，增加期待
4. **订阅引导** - 简洁明了，强调AI焦点

### ❌ 避免

1. **冗长的分段声明** - v2.0的17行三段式已被精简
2. **模糊的时间表述** - "本周"、"最近"等
3. **过多的下期预告** - 超过5项会显得杂乱
4. **营销性过强的语言** - 保持专业性

### 🆕 v3.0 变更重点

1. **篇幅大幅精简**：从17行减少到4行（-76%）
2. **保留法律合规**：投资免责声明依然完整
3. **AI焦点强化**：订阅提示强调"AI科技新闻"
4. **单段式设计**：提升阅读流畅度
5. **中文标点符号**：所有中文内容使用中文标点
